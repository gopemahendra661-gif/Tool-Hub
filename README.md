# My Tool Hub (mytoolhub.info)

The ultimate collection of 50+ free online tools. Designed for developers, writers, and everyone in between. Fast, secure, and easy to use.

## 🚀 Features

- **AI Tools:** Content Writer, YouTube Script Generator, Instagram Caption Generator, and more.
- **Text Tools:** Word Counter, Case Converter, Duplicate Line Remover, etc.
- **Image Tools:** Image Resizer, Compressor, Format Converter, Base64 Converter.
- **Developer Tools:** JSON Formatter, Regex Tester, UUID Generator, etc.
- **Calculators:** BMI, EMI, GST, Age, and more.
- **Privacy First:** Most tools process data locally in your browser.

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite, Tailwind CSS, Motion (Framer Motion).
- **Backend:** Express, Node.js.
- **Database:** Supabase (for feedback and ratings).
- **AI Integration:** OpenRouter (Gemini, Claude, GPT fallbacks).
- **Icons:** Lucide React.

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/mytoolhub.git
   cd mytoolhub
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file based on `.env.example` and add your API keys.

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```
